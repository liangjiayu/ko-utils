## 特性

- `ts`项目，良好代码提示
- 支持`npm`，`cdn`引用，支持独立模块



## 安装

### npm

```
npm i ko-utils-tiny
```



```javascript
import * as ko from 'ko-utils-tiny' // 全部引用

import { uniq } from 'ko-utils-tiny' // 引用对应的模块
```



### CDN

```html
<script src="ko-utils-tiny.js"></script>

<script>
  console.log(KO)
</script>
```
